KETCube Voltage Detector init.

Note
Some KETCube parts are not operationalunder certain voltage. E.g. HDC1080 is not opperational below 2V7 (HDC2080 will be used in next platform release, thus 1V62 will be the minimal voltage level for RHT sensor).
Below 1V9 is realy "dangerous" voltage, thus PWR_PVDLEVEL_0 is used as a default value
